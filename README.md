# SDN Mininet POX Project

## 📌 Description
This project demonstrates the implementation of **Software Defined Networking (SDN)** using **Mininet** and the **POX Controller**.  
It also includes **bandwidth analysis** between hosts using `iperf`.

---

# 🖧 Network Topology

- **1 Switch** → `s1`
- **3 Hosts** → `h1`, `h2`, `h3`

# 🚀 Technologies Used

- Python
- Mininet
- POX Controller
- OpenFlow Protocol
